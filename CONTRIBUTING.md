# Contributing to Lumen Argus

Thank you for your interest in contributing! This guide applies to all repositories under the [lumen-argus](https://github.com/lumen-argus) organization.

## Getting Started

1. Fork the repository
2. Create a feature branch from `main`
3. Make your changes
4. Run the test suite: `pytest`
5. Submit a pull request

## Development Setup

```bash
git clone git@github.com:lumen-argus/lumen-argus.git
cd lumen-argus
pip install -e ".[dev]"
pytest
```

## Pull Requests

- Keep PRs focused on a single change
- Include tests for new functionality
- Update documentation if needed
- Follow existing code style and conventions

## Reporting Issues

Use [GitHub Issues](https://github.com/lumen-argus/lumen-argus/issues) for bug reports and feature requests. Please check existing issues before creating a new one.
