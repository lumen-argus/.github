# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any Lumen Argus project, please report it responsibly.

**Do not open a public issue.**

Instead, please email **security@lumen-argus.dev** or use [GitHub's private vulnerability reporting](https://github.com/lumen-argus/lumen-argus/security/advisories/new).

We will acknowledge receipt within 48 hours and aim to provide a fix within 7 days for critical issues.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |

## Scope

This policy applies to all repositories under the [lumen-argus](https://github.com/lumen-argus) organization.
